title: The One Minute Server

Welcome to The One Minute Server

The Idea

The idea is to build something with node.js. And though i am fit with
database, i choosed a approach without a server-side database. I
decided to stored objects, just for one minute. What can you do with such a server ?
Let´s discover it.

Setup

You need a node-server running. And you can send Get-requests to the
server with objects in the url. e.g. : http://www.server.com/img/d=Base64Coded-Entity.
And you can get available data from the server: http://www.server.com/getdata

Server

Why “img” as path ? Because the first example dependes on
web-tracking, and it is so that you request a pixel from the server with
your tracking-request for most tracking solutions. So i went the same way.

The Img.js

I build a small tracking-library which you can integrate in your website to track some
features. The lib has on function which is “track(JSON-Object)”.
